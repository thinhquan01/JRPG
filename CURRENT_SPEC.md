# CURRENT_SPEC.md

## Current project
Tiny SNES-style JRPG scaffold as a proof-of-quality browser demo.

## Current scope
- tile-based overworld
- player movement
- one small town/outdoor map
- one indoor map/interior
- map transition between them
- 2 to 3 NPCs with simple dialogue
- one interactable object
- one simple menu screen
- one mock encounter or battle transition into a very simple battle screen
- ability to return from battle screen to overworld
- start state, playable loop, and restart

## Quality bar
- responsive controls
- coherent UI
- consistent visual style
- small scope
- no fake future features
