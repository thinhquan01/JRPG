# GAME_VISION.md

## Chosen direction
Build a **tiny first-person voxel sandbox proof of concept** first.

This is the core identity for now:
- small 3D sandbox
- editable voxel terrain/blocks
- every placed block obeys support/gravity rules
- some block types support visible orientation/rotation
- toybox first, RPG layer later if the sandbox is genuinely fun

## Explicit non-goals for this phase
- no open world
- no infinite terrain
- no quests
- no NPCs
- no combat
- no survival systems
- no crafting
- no multiplayer
- no engine work beyond what the POC needs

## Why this direction
This path best matches the strongest hook found so far:
- build / break / test / watch consequences
- world systems first
- fun with low content
- can later accept action or RPG layers without losing the core identity
