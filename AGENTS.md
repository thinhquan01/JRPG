# AGENTS.md

## Repo purpose
This repo is for iterating a tiny browser-based JRPG scaffold and related prototype passes.

## Rules
- Write real files, not code pasted in chat.
- Keep scope tiny unless explicitly expanded.
- Do not claim features exist unless they are implemented and verified.
- After each pass, update `TEST_NOTES.md` and `NEXT_TASK.md`.
- If blocked, reduce scope and still ship a smaller working pass.
- Prefer one coherent pass over many tiny no-op changes.
- Do not redesign unrelated systems when working on a focused task.

## Reporting format
Reply with:
- changed files
- what was done
- what still needs testing
- updated next task
